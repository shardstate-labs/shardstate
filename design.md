# SHARDSTATE - Design System

## Tone

- Dark
- Oppressive
- Urban decay
- System-controlled world
- Divided zones
- Functional, not comfortable

## Core Theme

- Humanity is exploited through PULSOS.
- PROTOCOL represents government control and invisible system authority.
- Clans represent different survival strategies.
- TITANS are anomalous and visually disconnected from the system.

## Visual Identity

- Dark matte surfaces.
- Industrial + organic fusion.
- Minimal glow used for information hierarchy, not decoration.
- No generic sci-fi dashboards.
- No one-note neon palettes.
- No decorative orbs/blobs.

## Clan Identity

- NEXUS `#6B5CE7`: cyberpunk street augmentations, illegal implants, scrap armor, purple/blue under-skin glow.
- TIDECALL `#00B4D8`: ocean nomads, bioluminescent tattoos, waterproof clothing, coral/shell adornments.
- ASHBORN `#FF6B35`: pyromaniac religious cult, burn scars, ember robes, ash-covered skin.
- ERRVOID `#B5179E`: nihilist hackers, glitch patterns, void portals, dark hooded figures.
- VAULT `#F7B731`: post-apocalyptic aristocracy, bunker luxury, gold, pre-fracture relics.
- MYCELIUM `#2ECC71`: human-fungal fusion, spores, bark skin, nature reclaiming humanity.
- IRONPACT `#E74C3C`: tribal mercenaries, salvaged armor, war paint, ruin-made weapons.
- SYNTHOS `#3498DB`: techno-utopian scientists, lab coats, drones, holograms, controlled optimism.
- LOOPKIN `#9B59B6`: time-displaced warriors, clock motifs, temporal afterimages.
- PHANTOM `#1ABC9C`: shadow operatives, cloaking, teal energy lines, surveillance drones.
- FREQUENZ `#F39C12`: post-apocalyptic artists, LED clothing, instruments as weapons, controlled chaos.
- PROTOCOL `#7F8C8D`: government remnants, pressed uniforms, rank symbols, maintained weapons.
- TITANS `#FFFFFF` + `#FFD828`: alien, godlike, non-human, visually wrong, less structured than all other clans.

Each clan must feel distinct in silhouette, material, palette, and mood.

## Cards

- Base cards: 12 per main clan.
- Special cards: 10 TITANS cards.
- Information priority:
  1. Stats
  2. Clan identity
  3. Ability text
  4. Clan bonus text
- Grand and Eco are visible type badges.
- TITANS is a clan, not a card type badge.

## Combat UI

- Instant feedback.
- Clear damage resolution.
- No ambiguous waiting states.
- Opponent cards may be inspected.
- Timers should create pressure without hiding information.
- Surrender and critical actions use in-app modals.

## Gamehub UI

- Prioritize utility and scanning.
- Collection, deck, market, packs, profile, and Battle Pass should feel like operational tools.
- Avoid marketing-style hero layouts inside the app.

## Localization

- Game-facing combat UI supports Spanish and English.
- Keep terms consistent:
  - Round = Ronda
  - Pulse = Pulsos
  - Clash = Combate
  - Surrender = Rendirse

## Anti-Patterns

- Generic sci-fi UI.
- Overused neon gradients.
- Clean corporate dashboards.
- Repetitive clan visuals.
- Browser-native alerts for important game actions.
- Hidden PvP state.

## Goal

The UI should feel like a controlled system extracting value from players: readable, sharp, oppressive, and built for function.
